About This Project

Washington DC has 103+ different permits spread across 16 different agencies, each with its own application portal, fee structure, and process. There is no single place to find them all.

A restaurant owner might need to navigate DOB, DC Health, ABCA, DDOT, FEMS, and DLCP — all separately, with no one telling them the full list up front. A homeowner wanting to renovate might discover mid-project that they need historic preservation review, an erosion control plan, and a public space permit they never knew existed.

This project is a proof of concept: an AI chatbot backed by a hand-curated database of every DC permit we could find. It's designed to answer the question "what do I need?" and point you to the right agency.

How It Works

When you ask a question, your query is matched against our permit database using semantic search (vector embeddings). The most relevant permits are retrieved and used as context for an AI model (Claude) to generate a natural language answer with specific permits, agencies, fees, and application links.

What It Costs

The entire stack runs on serverless AWS infrastructure for under $5/month. There's no database, no always-on servers, and no API Gateway. When nobody's asking questions, it costs essentially nothing.

Open Source

The permit database and all source code are open source on GitHub. If you spot a missing permit or incorrect information, please open an issue or PR.
